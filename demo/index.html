Integration

Two lines are enough in any project:

<!-- 1. Load the component -->
<script type="module" src="accessibility-menu.js"></script>

<!-- 2. Place the tag (ideally before </body>) -->
<accessibility-menu lang="en" position="bottom-right"></accessibility-menu>

Available Attributes

lang       "en" | "fr"                   (default: "en")
position   "bottom-right" | "bottom-left"
           "top-right"    | "top-left"   (default: "bottom-right")

From npm (local)

// ES module import
import './src/accessibility-menu.js';
